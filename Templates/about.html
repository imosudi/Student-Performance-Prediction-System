About the System

Student Performance Prediction System

This system uses a combination of fuzzy logic and regression techniques to predict student academic performance based on various factors.

Key Features
  • Predicts math, reading, and writing scores based on demographic factors
  • Uses fuzzy logic to model subjective factors like effort and motivation
  • Provides personalized recommendations based on predicted outcomes
  • Shows potential improvement compared to current scores
How It Works

The system uses two main components:

Regression Model

The regression component predicts scores based on:

  • Gender
  • Race/Ethnicity
  • Parental Education Level
  • Lunch Type (socioeconomic indicator)
  • Test Preparation Course Completion
Fuzzy Logic Component

The fuzzy logic component models subjective factors:

  • Attendance Rate
  • Study Hours Per Week
  • Motivation Level

These factors are used to adjust the base predictions from the regression model.

Dataset

This system was trained using datasets from:

  • UCI ML Repository - Student Performance Dataset
  • Kaggle - Students Performance in Exams
